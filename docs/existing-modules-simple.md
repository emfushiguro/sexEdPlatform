# Existing System Modules

Audit date: 2026-07-07

This document lists modules already present in the system in plain language.

## Fully Implemented Modules

### 1. Authentication and Account Access
Handles user sign up, login, logout, password reset, email verification, admin access, instructor access, and parent approval links.

### 2. User Profile and Profile Completion
Lets users complete and update profiles, change passwords, check username availability, and delete accounts.

### 3. Admin Dashboard
Gives admins main control panel, profile access, notifications, operational counts, and system shortcuts.

### 4. User Management and Roles
Lets admins create, view, edit, deactivate, delete, and manage users. Also handles roles, permissions, and user relationships.

### 5. Learning Content Management
Lets instructors and admins create and manage modules, lessons, topics, quizzes, quiz questions, images, and content ordering.

### 6. Module Review and Publishing
Lets instructors submit modules for review. Lets admins review, preview, approve, reject, archive, and apply penalties when needed.

### 7. Learner Modules and Progress
Lets learners browse modules, enroll, view lessons, complete topics, track progress, and finish modules.

### 8. Enrollment Management
Lets instructors and admins review learner enrollments, approve or reject requests, archive records, and manage parent approval for child enrollments.

### 9. Quizzes and Assessments
Lets learners take quizzes and view results. Lets instructors manage quizzes and view assessment activity.

### 10. Gamification
Handles points, streaks, shields, rewards, achievements, streak savers, and admin-controlled gamification rules.

### 11. Certificates
Lets learners earn, view, download, and verify certificates.

### 12. Subscriptions and Billing
Handles learner and instructor subscription plans, upgrades, renewals, cancellations, refunds, payment checkout, receipts, invoices, and payment status.

### 13. Module Purchases and Revenue
Handles paid module purchases, commissions, instructor earnings, admin revenue tracking, payout status, and financial reports.

### 14. Instructor Applications and Profiles
Lets learners apply to become instructors. Lets admins approve, reject, or archive applications. Lets instructors manage public profile details.

### 15. Parent and Child Management
Handles parent registration, child account creation, parent-child verification, child monitoring, child enrollment approval, and parent-child invitations.

### 16. Chat and Message Requests
Lets users start conversations, send messages, manage message requests, update chat status, mark messages as read, and report messages.

### 17. Reports, Suspensions, and Moderation
Handles learner reports, message reports, moderation cases, violations, suspensions, enforcement actions, appeals, and appeal conversations.

### 18. Connectors
Lets users discover, register, join, and manage connector groups. Includes connector approval, members, invitations, membership requests, roles, permissions, and subscriptions.

### 19. Seminars and Livestreams
Handles seminar browsing, registration, attendance, livestream access, comments, questions, speakers, registrants, connector-hosted seminars, and admin seminar moderation.

### 20. Notifications
Shows notifications for learners, instructors, and admins. Supports read status, dropdown alerts, and links back to related system activity.

### 21. Location Data
Provides region, province, city, and barangay selection data for profiles and registration flows.

### 22. Public Pages and APK Download
Includes landing page, privacy page, terms page, APK download, and QR code download access.

## Present But Not Fully Implemented

- Clinics, counselors, consultations, and organizations.
- Connector modules and connector educators pages.
